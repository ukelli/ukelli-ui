# API Pop

> Uke Pop 会判断用户的屏幕尺寸，如果超出显示范围，则切换到另一个方向显示

## Setup

```js
import { Pop } from 'ukelli-ui/core/popover'
```

## 构造一个新的 Pop

PopoverEntity 为弹出层对象的构造函数，可以通过 new 的方式来创建一个独特的弹出层

```js static
const pop = new PopoverEntity({
  id: '用于标识唯一性',
  fixed: {}, // 是否 fixed 定位
});

pop.show({
  width = 400, onClose,
  elem, // 相对的 document 元素
  children,
  props, // 传入 popover 实体对象的 props
});
```
