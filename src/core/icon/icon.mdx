# Icon

## FontAwesomeIcon

n 对应了 FontAwesomeIcon 的 icon-name，默认添加 `fa fa-` 前缀

<Playground>
  <Icon n="balance-scale" classNames={['ms10']} style={{fontSize: 20}} />
  <Icon n="allergies" classNames={['ms10']} style={{fontSize: 20}} />
  <Icon n="address-book" classNames={['ms10']} style={{fontSize: 20}} />
  <Icon n="address-card" classNames={['ms10']} style={{fontSize: 20}} />
</Playground>

- 如果需要切换图表库，可以通过 setUkelliConfig 设置 icon 的前缀
- 同时支持 icon 的名字映射，通过 setUkelliConfig 设置 iconMapper

-----------

## PureIcon

将 icon name 直接应用于元素，不添加前缀，不实用默认的 FontAwesomeIcon

<Playground>
  <PureIcon n="fas fa-dungeon" classNames={['ms10']} style={{fontSize: 20}} />
</Playground>

## 设置 IconMapper

```js
import { setUkelliConfig } from 'ukelli-ui';

// 以下为默认设置，可以覆盖
setUkelliConfig({
  iconMapper: {
    arrow: 'angle-left',
    date: 'calendar-alt',
    more: 'equals',
    close: 'times',
    noData: 'thermometer-empty',
    success: 'check',
    error: 'exclamation-circle',
    'circle-up': 'arrow-circle-up',
    'circle-down': 'arrow-circle-down',
  },
  // icon 的前缀, 关于 iconStyle 详情查看 https://fontawesome.com 描述
  iconPrefix: (iconStyle) => `fa${iconStyle} fa-`
});
```
