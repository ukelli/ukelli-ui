# InputSelector

> 选择器 & 输入控件

## Basic Usage

<Playground>
  <InputSelector
    values={{
      value1: 'value1',
      value2: 'value2',
      value3: 'value3',
    }}
    onChange={(inputVal, selectorVal) => {
      console.log(inputVal, 'inputVal')
      console.log(selectorVal, 'selectorVal')
    }} />
</Playground>
