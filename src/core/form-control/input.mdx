# Input

> 输入验证控件, 基础类提供基本验证

## Basic Usage

<Playground>
  <Input placeholder="请输入"
    onChange={e => console.log(e)}/>
</Playground>

## 固定的 Title

> 为 Input 添加 title，与 placeholder 不同，title 并不会在输入后消失

<Playground>
  <Input title="固定 Title" placeholder="placeholder"
    onChange={e => console.log(e)}/>
</Playground>

## 输出数字类型

<Playground>
  <Input title="输出 number 类型" placeholder="输出数字" outputType="number" 
    propsForInput={{
      type: 'number'
    }}
    onChange={e => console.log(e)}/>
</Playground>

## 过滤器

<Playground>
  <Input title="只能输入数字" filter={val => (val + '').replace(/[^0-9]/g, '')} />
</Playground>
