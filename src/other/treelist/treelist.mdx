## 调节 TreeListData 的字段与组件内部字段的对应关系

```js
<TreeList
  fieldMapper={{
    child: 'Children',
    id: 'ID',
    value: 'Value',
    title: 'Name',
  }}
  onChange={val => {
    console.log(val)
  }}
  treeData={treeData} />
```
