# Style

## Import Style

在项目入口 `.scss` 文件 `import` `ukelli-ui` 的内置样式

```scss
// 默认的颜色变量
@import 'ukelli-ui/style/color/default.scss';

// 依赖颜色变量的主样式入口
@import 'ukelli-ui/style/var.scss';
@import 'ukelli-ui/style/index.scss';
```

## 自定义样式变量

创建 `./var.scss` 变量 scss 文件，具体内容参考 https://github.com/ukelli/ukelli-ui/blob/master/style/color/default.scss

```scss
@import 'ukelli-ui/style/color/default.scss';

// 在 ukelli-ui/style/index.scss 之前覆盖 color 变量
@import './var.scss';

// 依赖颜色变量的主样式入口
@import 'ukelli-ui/style/index.scss';
```

### 3. 使用 color class

规则

1. bg_ 前缀设置背景颜色，bg_red
1. t_ 前缀设置字体颜色，t_green

```jsx
<div className="bg_red"></div>
```
