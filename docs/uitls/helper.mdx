# Helper

Ukelli 内置一些辅助函数，处理一些通用的情况

## 获取 Icons 的 mapper

```js
import { getIcon, getIconMapper } from 'ukelli/core/config'
```

## 动态加载脚本 LoadScript

```js
import { LoadStuff, LoadScript, LoadLink } from 'ukelli/core/utils'

// 通过动态标签加载资源
LoadStuff({ src, onload, type = script || css })

// 通过动态标签加载 script
LoadScript({ src, onload })

// 通过动态标签加载 css
LoadLink({ src, onload })
```
